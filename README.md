# ✨ Pretty Drafter

A browser-based whiteboard for teaching, studying and explaining ideas —
built for pen and geometry.

## Features

- Pen-based drawing (tablet & mouse friendly)
- Geometric primitives (line, rectangle, triangle, circle, trapezoid, parallelogram)
- Rotation and resize handles
- Layer support
- Keyboard shortcuts
- Designed for online teaching and explanations

## Input

- Works best with graphic tablets
- Fully usable with mouse and trackpad

## Keyboard shortcuts

- `P` — cycle pen tools
- `G` — cycle geometry tools
- `T` — cycle pen's thickness
- `C` — cycle pen's colors
- `1`-`3` - switching between layers
- `Esc` — exit geometry edit mode
- `Shift` — angle snapping (line, rectangle)
